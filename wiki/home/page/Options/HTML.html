Notes

  1. ^ Choose one of the following (default is unchecked)
    • checked : enables custom HTML page font
    • unchecked : disables custom HTML page font. The HTML page font will be the browser default (Arial 16)
  2. ^ Enter a font family name. (default is Arial)
  3. ^ Enter a number representing for a valid font size in pixels. (default is 16)
  4. ^ Enter a format for embedding the custom font info in the web page. (default is body, td {font-family:~{font_name};font-size:~{font_size}px;) }
