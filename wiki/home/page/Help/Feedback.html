Help/Feedback

From XOWA: the free, open-source, offline wiki application

Want to leave a comment? Report a defect? Try one of the following:

  • Post to XOWA's issue page: https://github.com/gnosygnu/xowa/issues . Note that you need to be a github user to post.
  • Post to my user talk page: http://en.wikipedia.org/wiki/User_talk:gnosygnu . If you are not a Wikipedia user, you will still be able to post messages (just like editing any other part of Wikipedia)
  • Send an email to gnosygnu at gmail dot com
