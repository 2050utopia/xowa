Setup notes

  • Download the latest version of jquery from https://gerrit.wikimedia.org/r/gitweb?p=mediawiki/core.git;a=tree;f=resources/jquery;h=8a58f7c31a0132f8c16b18b62cf5cc5f4aea5336;hb=master
  • Copy the following files to /xowa/bin/any/javascript/xowa/table_scripts/
    • jquery.js
    • jquery.makeCollapsible.js
    • jquery.tablesorter.js
  • Create a text file called /xowa/bin/any/javascript/xowa/table_scripts/xowa.ready.js
Save the most current contents of XOWA's xowa.ready.js file
